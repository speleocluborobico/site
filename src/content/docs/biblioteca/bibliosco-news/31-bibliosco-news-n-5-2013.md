Nel weekend del 23-24 marzo, passato in Liguria per un corso, ho avuto modo di recuperare presso il Gruppo Speleologico Savonese i numeri arretrati della loro rivista che non ricevevamo da un po’; inoltre l’amico Rinaldo ci ha fatto dono di un libro di cui vi parlo più sotto.  RIVISTE  **STALATTITI E STALAGMITI** (Periodico di Speleologia del Gruppo Speleologico Savonese) :  n° 22 – anno 1996;  n° 28 – anno 2002;  n° 29 – anni 2003-2004;  n°30 – anni 2005-2006;  n° 31 – anno 2007;  n° 32 – anno 2008  Gilberto Calandri del G.S. Imperiese ha invece fatto avere a tutti i partecipanti al corso una copia dell’ultimo numero del loro notiziario:  **BOLLETTINO** del Gruppo Speleologico Imperiese C.A.I. n° 62 – gennaio-dicembre 2010  Indice: vedere file pdf allegato  Infine, dall’altro gruppo di Savona, ci è giunto anche il:  **BOLLETTINO**  del Gruppo Grotte C.A.I. Savona n° 8 - 2011  che contiene anche un curioso esperimento con foto 3D e relativi occhialini per vederle nelle tre dimensioni.  Indice: vedere file pdf allegato  LIBRI  Come dicevo sopra Rinaldo Massucco, a nome del G.S.S., ci ha donato questa loro pubblicazione:  **“GROTTE E CARSISMO DELL'ALTA VAL BORMIDA - La Bormida di Millesimo”**  della collana "Grotte e Carsismo in Liguria".  Le riviste e il libro sopra descritti sono prelevabili per la consultazione presso la biblioteca sociale, facendo riferimento al regolamento di biblioteca per le modalità dei prestiti.  Buone letture e arrivederci alla prossima.  Max |  |  |  |  | |  | | | | | | | | | | |
